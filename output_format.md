# 出力フォーマット定義（初期ドラフト）

> AIが生成する報告書の構成・文体・Word出力仕様の案です。実際に使っている報告書のフォーマットに合わせて修正してください。

## 1. 出力物の種類

| 種類 | 用途 | 想定読み手 |
|---|---|---|
| 社外用報告書 | お客様・取引先へ提出する文書 | お客様、取引先の担当者 |
| 社内用報告書 | 社内共有・記録用の文書 | 上長、品質管理部門、関係部署 |

## 2. 社外用報告書

### 2.1 文体・トーン
- 丁寧語・敬語を用いる（お客様向けのビジネス文書として失礼のない表現）
- 専門用語・社内用語は避け、簡潔で分かりやすい表現にする
- お詫びの意を適切に含める（事実関係に基づき、誠実な印象を与える文面）
- 社内の原因分析・内部事情など、社外に開示すべきでない情報は含めない

### 2.2 構成案

```
1. 表題（例：ご報告とお詫び）
2. 宛名（お客様名／取引先名）
3. 挨拶文
4. 発生事象の説明（簡潔に）
5. お詫びの言葉
6. 対応内容・今後の対応予定
7. 締めの挨拶
8. 差出人（会社名・部署名・担当者名・連絡先）
9. 発行日
```

### 2.3 出力サンプルイメージ

```
ご報告とお詫び

株式会社〇〇 御中

平素より格別のお引き立てを賜り、厚く御礼申し上げます。

このたびは、〇〇（対象商品・サービス名）に関しまして、
ご迷惑・ご不便をおかけしましたこと、心よりお詫び申し上げます。

【事象の概要】
（クレームの概要を丁寧な文体で整形した内容）

【対応内容】
（現在の対応状況・今後の対応予定を整形した内容）

今後このようなことがないよう、再発防止に努めてまいります。
今後ともご愛顧賜りますよう、何卒よろしくお願い申し上げます。

〇〇株式会社 〇〇部
担当：〇〇
連絡先：〇〇

発行日：YYYY年MM月DD日
```

## 3. 社内用報告書

### 3.1 文体・トーン
- 簡潔・客観的な「である調」または社内標準の文体
- 事実・経緯・原因・対応・再発防止策を明確に区分して記載
- 推測部分は「推測」「暫定」であることが分かるように明記する

### 3.2 構成案

```
1. 表題（例：クレーム報告書（社内用））
2. 報告日／報告者
3. 基本情報（受付日、発生日、お客様名、対象商品・サービス名 等）
4. クレーム内容の詳細
5. 発生経緯
6. 原因分析（暫定含む）
7. 対応状況・対応予定
8. 再発防止策（暫定含む）
9. 影響範囲
10. 共有先・重要度
```

### 3.3 出力サンプルイメージ

```
クレーム報告書（社内用）

報告日：YYYY年MM月DD日
報告者：〇〇

【基本情報】
受付日：YYYY年MM月DD日
発生日：YYYY年MM月DD日
お客様名／取引先名：〇〇
対象商品・サービス名：〇〇

【クレーム内容】
（クレームの概要・発生経緯を整理した内容）

【原因分析（暫定）】
（入力された原因分析を整形した内容）

【対応状況・対応予定】
（現在の対応状況・今後の対応予定を整形した内容）

【再発防止策（暫定）】
（入力された再発防止策を整形した内容）

【影響範囲】
（入力された影響範囲の内容）

重要度／緊急度：〇
共有先：〇〇部、〇〇

以上
```

## 4. Word（.docx）出力仕様（案）

| 項目 | 内容（ドラフト） |
|---|---|
| ファイル形式 | .docx（Word形式） |
| フォント | 明朝体系（例：MS明朝）または会社標準フォント（要確認） |
| 用紙サイズ | A4、縦向き |
| 出力単位 | 社外用・社内用をそれぞれ別ファイルとして出力（もしくは1ファイル内で分けるかは要検討） |
| ファイル名規則（案） | `YYYYMMDD_クレーム報告書_社外用_お客様名.docx` / `YYYYMMDD_クレーム報告書_社内用.docx` |
| ロゴ・社印等 | 会社ロゴを社外用テンプレートに含めるか要検討 |
| テンプレート | 既存の社内Wordテンプレートがあれば、それをベースにすることが望ましい（要確認） |

## 5. 今後の検討事項

- [ ] 実際に使用している報告書のサンプル（実物、可能な範囲でマスキングしたもの）を入手し、構成・文言を合わせる
- [ ] 会社として決まった署名フォーマット・ロゴ使用ルールがあるか確認する
- [ ] 社外用・社内用を1つの操作で同時出力するか、個別に出力するかを決める
- [ ] Word出力時のレイアウト（余白・フォントサイズ等）の社内標準を確認する
- [ ] 生成された文面をアプリ内で編集してから出力できるようにするか検討する
